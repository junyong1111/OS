메모리를 보조하는 장치로 메모리와 다르게 현재

—# 시스템 버스

- 주소 버스 : 주소를 주고 받는 통로
- 제어 버스 : 제어 신호를 주고 받는 통로
- 데이터 버스 : 데이터와 명령어를 주고 받는 통로
